# Fortune Teller

A front-end JavaScript application that tells the user their daily fortune depending on the user's birthday.

### Technologies Used
- JavaScript ES5
- OOP
- HTML 5
- CSS 3
- Flexbox
- jQuery AJAX

### Live Demo
Try the application live at https://fortuneteller.helen-choi.com/
